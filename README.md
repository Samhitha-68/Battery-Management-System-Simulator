# Battery Management System (BMS) Simulator

## Overview
This project is a modular Battery Management System (BMS) simulator developed in C as part of my Embedded Systems learning journey.

## Features
- Battery structure using C struct
- Displays State of Charge (SoC)
- Displays State of Health (SoH)
- Displays Voltage
- Displays Current
- Displays Temperature
- Displays Charging Status

## Project Structure

```
main.c      -> Main program
battery.h   -> Battery structure and function declarations
battery.c   -> Battery-related functions
```
